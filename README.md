
# 🛍️Excel - Retail Sales Analysis Dashboard 

## 📌 Project Overview

This project is an interactive **Retail Sales Analysis Dashboard** built using **Microsoft Excel**. It provides valuable business insights by analyzing retail sales data through dynamic charts, KPIs, Pivot Tables, Pivot Charts, Slicers, and Maps.

The dashboard helps users monitor sales performance, profit trends, order statistics, category performance, regional sales, and top-selling products.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Track profit and profit margin.
- Monitor yearly growth.
- Compare sales and profit trends.
- Analyze category-wise sales and profit.
- Identify top-performing sub-categories.
- Visualize sales by region.
- Provide interactive filtering for better decision-making.

---

## 📊 Dashboard Features

### KPI Cards
- 💰 Total Sales
- 💵 Total Profit
- 📦 Total Quantity Sold
- 🛒 Number of Orders
- 📈 Profit Margin
- 📊 Year-over-Year (YoY) Growth

### Charts & Visualizations

- Sales vs Profit Analysis (Monthly)
- Category-wise Profit
- Category-wise Sales Share (Donut Chart)
- Sales by District (Map Chart)
- Top 5 Sub-Categories by Sales

### Interactive Filters (Slicers)

- Year
- Region
- Segment
